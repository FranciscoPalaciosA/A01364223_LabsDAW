Cuestionario 1: BD vs DBMS

Preguntas

¿Qué es entonces una base de datos - DB?

Una base de datos son muchos archivos que se relacionan entre ellos. Al momento de diseñar una debemos de ser cuidadosos en no duplicar datos.
Se podría decir que una base de datos es un sistema de archivos con directorio(s) que almacenan los archivos relacionados.
Sin embargo la diferencia entre ese sitema y una base de datos yace en el administrador de Base de Datos.

¿En qué casos es conveniente usar bases de datos?

Siempre que haya información en donde se relacionen diferentes archivos y que esa información quiera ser leída por varios usuarios o aplicaciones.

¿Qué es un sistema de gestión de base de datos - DBMS?

DBMS por sus siglas en inglés Database Management System, es aquél sistema formado por una colección de datos interrelacionados y varios programas para tener acceso a estos datos. Tiene como objetivo ser un ambiente conveniente y eficiente para poder extraer y guardar información (datos) a la base de datos.

Enlista y explica las funciones/responsabilidades que tienen los DBMS.

  • Independencia lógica y física de los datos
    • Poder hacer una modificación sin romper la estructura de la base.
  • Redundancia mínima
    • Evitar repeticiones innecesarias.
  • Acceso concurrente por parte de múltiples usuarios
    • Deben de poder atender a varios usuarios y aplicaciones al mismmo tiempo sin problema.
  • Distribución espacial de los datos
    • No es importante para el usuario en dónde se encuentren alojados los datos.
  • Integridad de los datos
    • Proteger la introducción de datos erróneos a la base de datos, tanto por errores físicos o de operación.
  • Resplado y recuperación
    • Se refiere a la capacidad de un sistema de base de datos de recuperar su estado en un momento previo a la pérdida de datos.
