Preguntas a responder

¿Por qué es una buena práctica separar el modelo del controlador?

Una de las principales razones es para poder reutilizar el código, si haces un buen modelo y un buen controlador, esas partes de código se pueden pasar facilmente de programa en programa.

¿Qué es SQL injection y cómo se puede prevenir?

Es un ataque a la base de datos en donde un usuario obtiene acceso a la misma y modifica la estructura. Cuando meten código SQL para modificar de cualquier forma la base de datos. Tener solo un usuario que pueda modificar la estructura de la base de datos
